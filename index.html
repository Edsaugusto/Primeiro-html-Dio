Heading Tags

As Heading Tags são usadas geralmente para titulos

No codigo elas sao colocar como h1, h2, h3, h4, h5 e h6 quanto maior a numeração menor tem a importancia do titulos

quando colocamos um texto no html dentro de uma tag h1, ele terá maior relevancia que o que está envolto de uma tag H2, por exemplo.

Tag de parágrafo

Quando estamos fazendo uma pagina com HTML utilizamos para fazer um paragrafo a tag p

Tags de lista

Nas tags de listas nós temos a ul, ol e li

Podemos utilizar essas tags para fazermo uma barra de navegações tambem

  • ul -> é uma lista desordenada, ou seja, ela nao tem uma ordem, geralmente vem com um ponto ao lado.
  • ol -> é a lista ordenada, utilizamos essa tag quando queremos fazer uma lista com uma ordem expecifica, podemos notar isso pois ao usa-la, ela vem enumerada
  • li -> o li é o listem item, ou seja, item da lista, utilizamos em conjunto com a ol ou ul.

as lista geralmente sao feitas dessa forma:

< ol >

< li >< /li >

< /ol >

ou

< ul >

< li >< /li >

< /ul >

tag para link

para colocamos um link nós utilizamos a tag a < a href="LINK"> o link < /a >

Com a tag a vem junto um atributo href que serve para colocarmos o link para qual queremos direcionar o internauta

Tags que alteram texto

ainda temos tags que alteram o texto, por exemplo o

< blockquote > utilizado para fazer uma citação < /blockquote >

temos a tag < i > para deixar em italico < /i >

a tag < b > para deixar em negrito < /b>

temos outras tags que modificam texto, sugiro pesquisar na internet para nao prolongarmos muito

Atributos das tags

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML
